== Description ==

The plugin adds a page for converting bitcoins into another currency. The plugin works with the CoinGate system, processes the callback, sends out email based on payment results.

== Installation ==

1. Upload the entire `cbtc_convert_bitcoin` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.

You will find 'Convert Menu' menu in your WordPress admin panel.
